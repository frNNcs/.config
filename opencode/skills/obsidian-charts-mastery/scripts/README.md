# Scripts

No mandatory executable script is required for initial version `0.1.0`.

Rationale:
- Core workflows are YAML authoring, table linkage, and DataviewJS transformations.
- Deterministic automation opportunities exist but are optional for first release.

Potential future scripts:
- YAML chart block validator (`validate-chart-yaml.*`)
- Bulk normalize chart indentation in vault notes
- Preflight checker for chart-from-table IDs across files
