# Example 04 — Doughnut distribution

```chart
type: doughnut
labels:
  - Product
  - Operations
  - Sales
  - Marketing
series:
  - title: Budget
    data:
      - 38
      - 25
      - 22
      - 15
width: 40%
labelColors: true
legendPosition: right
```

Why:
- Show compact part-to-whole distribution.
- Improve readability with label-color linkage.
